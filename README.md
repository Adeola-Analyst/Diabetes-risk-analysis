# Diabetes Risk Analysis
## Health Data Analytics Project

### Overview
This project presents a comprehensive analysis of the Pima Indians Diabetes Dataset containing medical records 
of 768 female patients. The goal is to identify key risk factors associated with diabetes using Python and data 
analytics tools.

### Dataset
- Source: UCI Machine Learning Repository
- Patients: 768 female patients
- Features: 9 clinical variables
- Target: Diabetes diagnosis (Yes/No)

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colaboratory

### Key Findings
- Diabetes prevalence: 34.90% of patients
- Strongest predictor: Glucose (correlation: 0.49)
- Second strongest: BMI (correlation: 0.31)
- Diabetic patients had 28% higher glucose levels
- Diabetic patients were on average 6 years older
